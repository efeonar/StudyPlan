# StudyPlan - Öğrenci Takip ve Çalışma Sistemi

Bu proje, öğrencilerin akademik süreçlerini takip edebildiği ve öğretmenlerin öğrencileri yönetebildiği uçtan uca bir web tabanlı platformdur.

## 🚀 Kullanılan Teknolojiler
* **Backend:** Native PHP
* **Veritabanı:** MySQL
* **Frontend:** HTML5, CSS3, JavaScript

## 📌 Temel Özellikler
* Öğretmen ve Öğrenci için ayrı giriş panelleri
* Özelleştirilebilir ders ve çalışma planı oluşturma
* Süreç takip ve başarı grafiklerinin izlenmesi
